Hello! Welcome to my Dog Breed Classifier

How does it work?

After you upload a picture and start the prediction process, first the program detects if the uploaded image has a human face or is a dog, if neither is found an error will be shown.


Image 1. Face or Dog Detector

if the uploaded picture has a human face or dog, the program will start the process of object recognition to predict the breeds according to the trained algorithm. This algorithm is a deep learning algorithm done with a Convolutional Neural Network (CNN).

It was trained with a total of 8,351 dog images from 133 dog categories (breeds), the data was divided using 6,680 dog images for training, 835 for validation and 836 for testing.

As an output a bar chart will be shown with the top 5 predicted breeds according to the characteristics found by the trained algorithm.

How to use the App

  1. Locate the Navigation Bar on top part of the page (Image 2)
  2. Click on the “Predict” button to change to the prediction page (Image 2)

    3. Image 2. Navigation Bar

  3. Click on the “Choose File” (1) button to select a picture from your local drive that contains the Face or Dog for prediction. (Image 3)
  4. Click on the “Load Image” (2) button to display it on the application, an information message with the legend “Image Uploaded!!!” will be shown. (Image 3)
  5. After the picture is shown on the application now we can click on the “Predict” (3) button, after the process finishes the bar chart with the predicted breeds and percentages will be shown. (Image 3)
  6. To classify another image repeat the process from step 3. (Image 3)
